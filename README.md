## 🚀 Run Locally

### 1. Clone the Repository

```bash
git clone <repository-url>
cd <project-folder>
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the App

```bash
npm start
```

or

```bash
npx expo start
```

### 4. Run on Device

- Install **Expo Go** on your Android/iPhone.
- Scan the QR code shown in the terminal/browser.
